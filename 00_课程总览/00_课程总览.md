# 第 00 章 课程总览

## 课程定位

这套资料只讲 AI 使用常识，不讲模型训练、数学公式和 API 编程细节。

| 项目 | 内容 |
| --- | --- |
| 适合 | 零基础用户、职场使用者、需要做工具选型的人 |
| 不适合 | 需要训练模型、微调模型、开发复杂 AI 系统的人 |
| 信息基准 | 2026 年 6 月 |
| 使用原则 | 先低风险试用，再在关键任务中建立验证流程 |

## 学习路径

| 目标 | 章节 |
| --- | --- |
| 快速理解 AI | 01 |
| 选择模型和工具 | 02、03 |

## 每章读什么

| 章 | 重点 |
| --- | --- |
| 01 | AI、大语言模型、常见误区 |
| 02 | 模型差异（通用/垂直、闭源/开源、国内/国外、规模、使用方式） |
| 03 | 能力梯队、价格、Token 估算、预算控制、性价比、选型规则 |
| 04 | 黑箱、可能性空间、信息分布、提示词模板、反馈调节、验证方法、分级控制 |

## 使用约定

- 价格、模型版本、上下文长度不写死；正式使用前查官方页面。
- 模型推荐只代表选型思路，不代表固定排名。
- 医疗、法律、金融、合规、人事、采购等高风险任务不能只依赖 AI 输出。

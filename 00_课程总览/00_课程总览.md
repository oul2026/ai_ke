## 课程定位

这套资料只讲 AI 使用常识，不讲模型训练、数学公式和 API 编程细节。

| 项目 | 内容 |
| --- | --- |
| 适合 | 零基础用户、职场使用者、需要做工具选型的人 |
| 不适合 | 需要训练模型、微调模型、开发复杂 AI 系统的人 |
| 信息基准 | 2026 年 6 月 |
| 使用原则 | 先低风险试用，再在关键任务中建立验证流程 |

## 学习路径

| 目标 | 章节 |
| --- | --- |
| 建立 AI 总地图 | 01 |
| 看懂大模型分类和差异 | 02 |
| 做能力、价格和预算判断 | 03 |
| 让 AI 按目标稳定工作 | 04 |

## 每章读什么

| 章 | 重点 |
| --- | --- |
| 01 | AI 总地图：传统 AI、机器学习、深度学习、生成式 AI、LLM；AI 对个人能力的放大和补充 |
| 02 | LLM 分类与差异化：语言、多模态、图像/音频/视频、Embedding；通用/垂直、闭源/开放权重、使用入口、规模架构、自部署关注点 |
| 03 | 能力价格与选型：模型梯队、输入/输出/缓存/思考价格、Token 预算、性能和性价比图 |
| 04 | 如何正确让 AI 工作：思维模型、控制模型、提示词结构、上下文管理、反馈迭代、验证与信任 |

## 使用约定

- 价格、模型版本、上下文长度不写死；正式使用前查官方页面。
- 模型推荐只代表选型思路，不代表固定排名。
- 医疗、法律、金融、合规、人事、采购等高风险任务不能只依赖 AI 输出。

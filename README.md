# AI 科普课程

> 一套面向**零基础成年人**的 AI 通识课程,以"思维导图作骨架,文字作详解"的方式,带你从"完全不懂"走到"熟练用 AI 解决问题"。

---

## 这门课适合谁

- 听说过 ChatGPT、文心、豆包、Claude,但说不清它们到底是什么
- 想在工作、学习、生活中真正用上 AI,却不知道从哪里下手
- 不想看公式和代码,只想搞懂"AI 是什么、能做什么、怎么挑、怎么用、怎么避坑"

## 你能学到什么

学完后,你能够:

1. 用一句话讲清楚 AI 和大模型是什么
2. 看懂市面上大模型的差异,知道哪款适合你
3. 理解"参数""API""自部署"这些概念
4. 用好提示词与上下文,让 AI 输出真正有用的内容
5. 识别 AI 的"一本正经胡说八道",不被它带偏
6. 把 WorkBuddy、CodeBuddy 等工具用成日常生产力

## 课程结构(13 章)

| 编号 | 章节 | 一句话介绍 |
| --- | --- | --- |
| 00 | 课程总览 | 看清整张地图,选好学习路径 |
| 01 | AI 是什么 | 一杯咖啡的时间,搞懂 AI 的本质 |
| 02 | AI 如何工作 | 用"超级鹦鹉"理解大模型的原理 |
| 03 | 大模型能做什么 | 文字、图像、语音、视频、代码……的能力清单 |
| 04 | 大模型的差异化 | 通用 vs 垂直、闭源 vs 开源、国内外 |
| 05 | 大模型的能力梯队 | 一梯队、二梯队、三梯队代表与选型建议 |
| 06 | 智商与参数的关系 | 为什么"参数越大越聪明"是误区 |
| 07 | API 与自己部署 | 两种使用方式的区别与适用人群 |
| 08 | 大模型的成本与定价 | Token 怎么算、各模型多少钱、怎么省钱 |
| 09 | AI 的应用场景 | 办公、学习、创作、生活、专业领域 |
| 10 | WorkBuddy 与 CodeBuddy | 腾讯的 AI 助手,口语中也叫"龙虾" |
| 11 | 提示词与上下文工程 | 让 AI 输出你想要的结果的实战技巧 |
| 12 | AI 幻觉与验证 | 识别"胡编乱造",守住最后一道防线 |
| 13 | 总结与展望 | 趋势、学习路径、持续精进建议 |

## 怎么用这套资料

每个章节目录下有两个文件:

- `XX_章节名.md` —— 详细讲解,建议从头读到尾
- `XX_章节名.xmind` —— 思维导图,适合复习、分享、打印贴在墙上

**推荐学习顺序**:

```
看导图建立框架 → 读 md 补充细节 → 再看导图回顾
```

## 阅读建议

- 每章约 12-15 分钟阅读量,13 章总计约 3.5 小时
- 不必一次读完,每天 1-2 章,持续一周就能通盘掌握
- 学完一章后,挑一两个身边的实际任务去试着用 AI 完成,效果最好

## 关于工具

课程中提到的 AI 工具(如 WorkBuddy、CodeBuddy)均为举例,目的是帮你建立认知。具体使用时请以官方最新版本为准。

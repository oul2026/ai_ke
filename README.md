# AI 使用速查课

面向零基础用户的 AI 通识资料。目标不是介绍行业故事，而是回答四个问题：

1. AI 是什么，边界在哪里
2. 模型怎么选，成本怎么算
3. 工作中怎么用，提示词怎么写
4. 输出怎么验证，哪些内容不能直接信

## 章节

| 编号 | 主题 | 重点 |
| --- | --- | --- |
| 00 | 课程总览 | 学习路径与使用约定 |
| 01 | AI 是什么 | 定义、误区、产品家族 |
| 02 | 大模型的差异化 | 通用/垂直、闭源/开源、国内/国外 |
| 03 | 能力和价格 | 梯队、价格、Token 估算、预算方法 |
| 04 | 如何正确让 AI 工作 | 提示词、反馈调节、验证方法 |

## 使用方式

- 想快速上手：读 01、04。
- 想做选型：读 02、03、04。
- 想系统学习：从 00、01、02、03、04 顺序读。

## 约定

- 模型能力、价格、上下文长度变化很快，正式选型以官方页面为准。
- 文中产品只作类型示例，不作长期排名。
- 关键事实、数字、引用、医疗法律金融建议必须核验。

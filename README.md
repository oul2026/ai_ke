# AI 使用速查课

面向零基础用户的 AI 通识资料。主线按“总图 → LLM → 选型 → 使用”的顺序展开：

1. 先建立 AI 总地图，分清传统 AI、机器学习、深度学习、生成式 AI 和 LLM。
2. 再进入 LLM，按模型类型、能力边界、开放方式、使用入口和部署成本拆开看。
3. 然后进入能力与价格，解决“选哪个、多少钱、怎么预算”的问题。
4. 最后落到使用方法，用思维模型、控制模型和验证方法让 AI 输出可控。

# 第 08 章 大模型的成本与定价

> 用 AI 到底花多少钱？Token 是什么？为什么同一段话，不同模型的价格差 10 倍？这一章帮你算清楚账。

---

## 8.1 什么是 Token

Token 是大模型计费的**最小单位**，可以理解为"AI 阅读和输出文字时切出来的小碎片"。

### Token 不等于字数

| 语言 | 大致换算关系 |
| --- | --- |
| 英文 | 1 个 Token ≈ 4 个英文字符 ≈ 0.75 个英文单词 |
| 中文 | 1 个汉字 ≈ 1.5 ~ 2 个 Token（因模型而异） |

**举个例子：**

- "你好世界" → 大约 4~6 个 Token
- "Hello world" → 大约 2 个 Token
- 一篇 1000 字的中文文章 → 大约 1500~2000 个 Token

> 💡 **简单记法**：中文场景下，**1 个字 ≈ 1.5 个 Token**，够用了。

### 为什么 Token 很重要

- 你发给 AI 的内容（输入/Prompt）按 Token 计费
- AI 回复你的内容（输出/Completion）也按 Token 计费
- 一次对话的总费用 = 输入 Token 费 + 输出 Token 费

---

## 8.2 输入与输出的价差

大多数 API 服务中，**输出比输入贵 2~4 倍**，原因是生成内容比理解内容消耗更多算力。

| 项目 | 说明 |
| --- | --- |
| 输入 Token | 你发给 AI 的提示词、上下文、系统指令等 |
| 输出 Token | AI 生成的回复内容 |
| 价差原因 | 生成需要逐字推理，计算量更大 |

> 💡 有些平台还有**缓存 Token**（Cached Token）的概念：如果多次对话中部分内容相同，缓存部分的价格会便宜很多。

## 8.4 算一笔实际的账

**场景：每天用 AI 写 5 篇 800 字的工作日报**

假设每篇日报：
- 输入（你的要求 + 上下文）：约 500 字 ≈ 750 Token
- 输出（AI 生成的日报）：约 800 字 ≈ 1200 Token
- 每天 5 篇 → 输入 3750 Token + 输出 6000 Token

**月用量（22 个工作日）：**
- 输入：约 82,500 Token ≈ 0.083 百万 Token
- 输出：约 132,000 Token ≈ 0.132 百万 Token

| 模型 | 月费用估算 |
| --- | --- |
| GPT-5.5 | 约 $0.21 + $1.98 = **$2.19（约 ¥16）** |
| GPT-5.5 mini | 约 **$0.02 + $0.26 = $0.28（约 ¥2）** |
| DeepSeek-V4 | 约 **$0.02 + $0.06 = $0.08（约 ¥0.6）** |
| 豆包 Pro | 约 **¥0.3** |

> 💡 **结论**：日常轻度使用，大多数模型**每月花费不超过一杯咖啡**。选对模型很重要。

## 8.5 免费额度与省钱技巧

### 免费额度

很多平台提供新用户免费额度或每日免费调用次数：

- **OpenAI**：新注册账号有 $5 免费额度（有效期 3 个月）
- **智谱AI**：新用户注册得 2000 万 Token 免费额度
- **百度文心**：Speed 等轻量模型长期免费
- **阿里通义**：新用户有百万 Token 免费额度
- **Google Gemini**：Flash 版本有每分钟请求次数限制的免费层
- **豆包**：新用户有免费体验额度

### 省钱技巧

1. **选对模型等级**：日常对话不需要旗舰模型，用 mini/turbo/flash 版本就够了
2. **精简输入**：不要把整篇文章都丢给 AI，只给必要的上下文
3. **善用缓存**：重复对话中保持系统提示词不变，利用缓存 Token 降价
4. **批量处理**：把多个小任务合并成一次请求，减少重复的输入 Token
5. **控制输出长度**：在提示词中明确"用 200 字以内回答"，避免 AI 啰嗦
6. **关注优惠活动**：各平台经常有限时降价或充值赠送

## 8.6 API 计费 vs 订阅制

| 方式 | 说明 | 适合谁 |
| --- | --- | --- |
| **API 按量付费** | 用多少 Token 付多少钱 | 开发者、企业、需要集成到自己系统的用户 |
| **订阅制（月费）** | 如 ChatGPT Plus $20/月、Claude Pro $20/月 | 普通用户，日常在网页/App 上对话 |
| **免费使用** | 有次数或功能限制 | 偶尔用一下的用户 |

> 💡 **如果你只是在网页上聊天**，订阅制最省心；**如果你要做自动化或开发应用**，API 按量付费更灵活。

## 本章小结

| 核心概念 | 一句话记住 |
| --- | --- |
| Token | AI 计费的"字"，中文 1 字 ≈ 1.5 Token |
| 输入 vs 输出 | 输出通常比输入贵 2~4 倍 |

| 省钱关键 | 选对模型 + 精简输入 + 控制输出长度 |

# 第 10 章 WorkBuddy 与 CodeBuddy

> 腾讯出品的 AI 助手，在腾讯内部口语中也叫"龙虾"。这一章帮你快速上手这两个工具。

---

## 10.1 WorkBuddy 是什么

WorkBuddy 是腾讯推出的**办公 AI 助手**，定位是帮职场人士提高工作效率。

### 核心能力

| 能力 | 说明 |
| --- | --- |
| **文档写作** | 帮你写邮件、报告、方案、PPT 大纲 |
| **文档阅读** | 上传长文档，AI 帮你提取要点和总结 |
| **数据分析** | 处理表格数据，生成图表和结论 |
| **会议纪要** | 把会议录音转成文字并整理纪要 |
| **翻译** | 支持多语言文档翻译 |
| **日程管理** | 辅助安排会议和待办事项 |

### 在哪里使用

- 腾讯会议（集成）
- 企业微信（集成）
- 独立网页版
- 移动端 App

### 适合谁

- 日常办公白领
- 需要频繁写文档的岗位（行政、市场、运营）
- 腾讯生态用户（企业微信、腾讯会议重度使用者）

---

## 10.2 CodeBuddy 是什么

CodeBuddy 是腾讯推出的**编程 AI 助手**，定位是帮开发者提高编码效率。

### 核心能力

| 能力 | 说明 |
| --- | --- |
| **代码生成** | 根据描述生成代码片段 |
| **代码补全** | 写代码时实时补全下一行 |
| **Bug 修复** | 帮你定位和修复代码错误 |
| **代码解释** | 把复杂代码翻译成自然语言 |
| **单元测试** | 自动生成测试代码 |
| **代码重构** | 优化代码结构和性能 |

### 支持的语言

- Python、Java、JavaScript、TypeScript
- C/C++、Go、Rust
- SQL、HTML/CSS
- 以及更多主流编程语言

### 在哪里使用

- **VS Code 插件**：最推荐，在编辑器里实时辅助
- **JetBrains 插件**：IntelliJ、PyCharm 等
- **独立网页版**：在线对话式编程

### 适合谁

- 专业程序员
- 学习编程的新手
- 需要写简单脚本的非技术岗位

## 10.3 WorkBuddy 使用技巧

### 技巧一：给出足够的上下文

**差的提示**：
> "帮我写个邮件"

**好的提示**：
> "帮我写一封邮件，发给客户张经理，主题是项目进度延期通知。原因是供应商延迟交货，预计延期 2 周。语气要专业但不要太正式。"

### 技巧二：利用模板化能力

对于重复性高的文档（周报、日报、固定格式报告），让 AI 记住格式：
> "以后每次我列出本周工作，你都按照以下格式整理：1. 本周完成 2. 进行中 3. 下周计划 4. 风险与问题"

### 技巧三：善用文档阅读功能

收到一份 50 页的报告？上传后问 AI：
> "这份报告的核心结论是什么？列出 5 个要点"
> "报告中提到的三个最大风险是什么？"

---

## 10.4 CodeBuddy 使用技巧

### 技巧一：描述清楚需求

**差的提示**：
> "写一个排序函数"

**好的提示**：
> "用 Python 写一个函数，输入一个字典列表，按照 'age' 字段从小到大排序。如果 age 相同，按 'name' 字段字母顺序排。"

### 技巧二：让 AI 解释代码

接手了一个老项目，看不懂代码？直接选中代码问：
> "这段代码在做什么？用非技术人员能理解的方式解释"

### 技巧三：让 AI 写测试

写完一段代码后：
> "给这个函数写单元测试，包括正常情况、边界情况和异常情况"

## 10.5 WorkBuddy vs 其他 AI 助手

| 维度 | WorkBuddy | ChatGPT | 文心一言 |
| --- | --- | --- | --- |
| 集成生态 | 腾讯系（企微、腾讯会议） | 独立 + 插件 | 百度系 |
| 办公深度 | 深度集成企业办公工具 | 通用对话 | 通用对话 |
| 数据安全 | 企业级安全合规 | 数据可能用于训练 | 国内合规 |
| 中文优化 | 好 | 好 | 好 |
| 价格 | 企业定价 | $20/月（Plus） | 部分免费 |

> 💡 如果你的公司已经在使用腾讯生态（企业微信、腾讯会议），WorkBuddy 的集成优势非常明显。

## 10.6 上手第一步

不知道从哪里开始？现在就试：

### WorkBuddy 入门任务

1. 打开 WorkBuddy（网页版或企微集成）
2. 输入："帮我写一封请假邮件，请假原因是家里有急事，请假时间是明天一天"
3. 看看生成的结果，尝试修改和追问

### CodeBuddy 入门任务

1. 在 VS Code 中安装 CodeBuddy 插件
2. 新建一个 Python 文件，写一行注释：`# 读取 CSV 文件并统计每列的平均值`
3. 看 CodeBuddy 自动补全的代码

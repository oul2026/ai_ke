# 第 03 章 大模型能做什么

> AI 不只会聊天。这一章带你看看它的"能力清单"，以及它暂时还做不到的事。

---

## 3.1 文字能力：AI 最强的领域

大模型最擅长的就是处理文字，这也是它的"看家本领"。

### 能做的事

| 能力 | 举例 | 效果 |
| --- | --- | --- |
| **写作** | 写邮件、周报、文案、故事、诗 | 速度快，质量中等，需要人工润色 |
| **改写润色** | 把口语化的文字改成正式文体 | 非常好用，是日常高频场景 |
| **翻译** | 中英互译，支持几十种语言 | 日常翻译接近专业水平 |
| **总结提炼** | 把长文浓缩成要点 | 节省大量阅读时间 |
| **解释概念** | 用通俗的话解释专业术语 | 像随时在线的"百科全书" |
| **头脑风暴** | 列出 10 个产品名字候选 | 数量多，质量参差不齐 |
| **角色扮演** | 假装是面试官、历史人物 | 适合练习和模拟场景 |

### 做不到的事

- **原创性深度思考**：AI 能模仿，但不能真正"创新"
- **100% 准确的事实**：AI 会编造数据（后面会讲"幻觉"）
- **理解你的真实意图**：它只是根据你的文字来推测

---

## 3.2 图像能力：AI 的"第二技能"

除了文字，AI 还能处理图像。这属于**多模态能力**（同时处理多种信息形式）。

### 图像生成

用文字描述，AI 帮你画出来：

| 工具 | 特点 |
| --- | --- |
| GPT Image（OpenAI） | GPT-5 内置图像生成，对话式创作 |
| Midjourney | 艺术风格突出，社区活跃 |
| Stable Diffusion | 开源免费，可本地运行 |
| 通义万相（阿里） | 中文理解好，免费可用 |

**使用场景**：
- 产品概念图、社交媒体配图、PPT 插图
- 室内设计效果图参考
- 创意海报初稿

### 图像理解

给 AI 看一张图，它能：
- 描述图片内容
- 读取图片中的文字（OCR）
- 分析图表和数据
- 识别物体和场景

> 💡 GPT-5、Claude Opus 4.8 等都支持“看图说话”，你可以直接把截图发给它们分析。

---

## 3.3 语音能力：能听能说

### 语音识别（听）

- 把语音转成文字
- 支持多语言、方言
- 代表产品：讯飞听见、Whisper（OpenAI）

### 语音合成（说）

- 把文字变成自然语音
- 可以克隆声音（需谨慎使用）
- 代表产品：ChatGPT 语音模式、讯飞配音

### 实时对话

- 像打电话一样和 AI 对话，不用打字
- ChatGPT 的"语音模式"已经非常流畅

---

## 3.4 视频能力：正在快速发展

### 视频生成

| 工具 | 现状 |
| --- | --- |
| 可灵（快手） | 国产代表，效果不错 |
| Runway | 已商用，可生成短视频 |
| Pika | 简单易用，适合短视频创作 |

### 视频理解

- 上传视频，AI 分析内容
- 目前能力有限，主要靠"截图+文字描述"组合处理

> 💡 视频能力是 AI 发展最快的领域之一，但目前还不够成熟，生成质量和一致性有待提升。

---

## 3.5 代码能力：程序员的得力助手

AI 在编程方面的能力已经相当强：

| 能力 | 说明 |
| --- | --- |
| 写代码 | 根据描述生成代码片段或完整程序 |
| 解释代码 | 把代码翻译成人能懂的语言 |
| 改 Bug | 帮你找出并修复代码错误 |
| 重构优化 | 让代码更简洁、更高效 |
| 跨语言翻译 | 把 Python 代码转成 JavaScript 等 |

**代表性编程 AI 工具**：
- **GitHub Copilot**：集成在编辑器里，实时辅助写代码
- **CodeBuddy**（腾讯）：国产编程助手
- **Cursor**：基于 AI 的新一代代码编辑器

> 💡 即使你不是程序员，也可以让 AI 帮你写简单的 Excel 公式、HTML 网页、数据处理脚本。

---

## 3.6 AI 暂时做不到的事

知道 AI 的**能力边界**比知道它能做什么更重要：

| 做不到的事 | 原因 |
| --- | --- |
| 保证事实 100% 准确 | 模型基于概率生成，会"编造"信息 |
| 理解实时世界 | 训练数据有截止日期，不知道"今天"发生了什么（除非联网搜索） |
| 做真正的逻辑推理 | 擅长模式匹配，但复杂多步推理容易出错 |
| 替代专业判断 | 不能替代医生诊断、律师法律意见 |
| 拥有个人经验 | 它没有亲身经历过任何事 |
| 保守秘密 | 对话内容可能被用于训练（取决于平台政策） |

---

## 3.7 能力总结表

| 能力类别 | 成熟度 | 代表工具 | 适合场景 |
| --- | --- | --- | --- |
| 文字 | ★★★★★ | ChatGPT、文心、Claude、DeepSeek | 日常办公最高频 |
| 图像生成 | ★★★★☆ | GPT Image、Midjourney、可灵 | 设计参考、配图 |
| 图像理解 | ★★★★☆ | GPT-5、Claude Opus 4.8 | 看图分析、OCR |
| 语音 | ★★★★☆ | 讯飞、ChatGPT语音 | 转录、配音 |
| 视频 | ★★★☆☆ | 可灵、Runway、Pika | 创意预览 |
| 代码 | ★★★★☆ | Copilot、CodeBuddy | 编程辅助 |

---

## 本章小结

| 核心要点 | 一句话记住 |
| --- | --- |
| 最强能力 | 文字处理——写、改、翻译、总结 |
| 多模态 | 图像、语音、视频能力都在快速发展 |
| 代码 | 程序员的强力助手，非程序员也能用 |
| 能力边界 | 不保证准确、不懂实时世界、不能替代专业判断 |

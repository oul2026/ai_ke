# XMind 流程图（relationship 连接线）

```text
逻辑图
  │
  ▼
明确目标
  │
  ▼
约束条件/缩小空间
  │
  ▼
观察反馈
  │
  ▼
达标？
  ├── 是 ──► 明确目标
  └── 否 ──► 约束条件/缩小空间
```

流程边：
1. 明确目标 -> 约束条件/缩小空间
2. 约束条件/缩小空间 -> 观察反馈
3. 观察反馈 -> 达标
4. 达标 -> 是
5. 达标 -> 否
6. 否 -> 约束条件/缩小空间
7. 是 -> 明确目标
8. 逻辑图 -> 明确目标

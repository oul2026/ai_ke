# 第 05 章 大模型的能力梯队

> 市面上有几十款大模型，到底谁强谁弱？这一章帮你建立一个清晰的"等级认知"，知道什么场景用什么模型。

---

## 5.1 三个梯队的划分

大模型可以大致分为三个梯队，每个梯队有不同的定位：

| 梯队 | 定位 | 类比 |
| --- | --- | --- |
| 一梯队 | 旗舰级，能力天花板 | 行业顶尖专家 |
| 二梯队 | 主力级，性价比最优 | 经验丰富的资深员工 |
| 三梯队 | 轻量级，速度快成本低 | 高效能的实习生 |

---

## 5.2 一梯队：旗舰级模型

这些模型代表了当前 AI 的最高水平，在复杂推理、创意写作、代码生成等难度高的任务上表现最好。

| 模型 | 出品方 | 核心优势 |
| --- | --- | --- |
| GPT-5.5 | OpenAI | 推理深度最强，支持计算机控制（Agent），1M+上下文 |
| Claude Opus 4.8 | Anthropic | 编码能力最强，1M上下文，多智能体协调 |
| Gemini 3.1 Pro | Google | 多模态综合最强，基准测试 16 项领先 13 项 |
| DeepSeek-V4 | 深度求索 | 已跻身第一梯队，性价比极高，中文场景强 |

### 什么时候用一梯队

- 需要深度分析和复杂推理
- 写长篇文章、专业报告
- 代码架构设计和复杂编程
- 需要最高质量输出的关键场景

### 代价

- **价格高**：比二梯队模型贵 10-50 倍（详见第 08 章）
- **速度慢**：生成回复的时间更长
- **额度限制**：免费版通常有严格的次数限制

---

## 5.3 二梯队：主力级模型

性能接近旗舰，但价格更友好。**日常使用最推荐这个梯队。**

| 模型 | 出品方 | 核心优势 |
| --- | --- | --- |
| GPT-5 mini | OpenAI | 性价比高，英文场景首选 |
| 通义千问 Qwen3.7-Max | 阿里 | 智能体原生设计，盲测国产第一，可自主执行35小时任务 |
| GLM-5.1 | 智谱AI | 代码能力突出，智能体能力领先 |
| Kimi K2.5 | 月之暗面 | 200万token超长上下文，文档处理专家 |
| 文心一言 5.0 | 百度 | 搜索增强，政务教育场景优势 |

### 什么时候用二梯队

- 日常写作、翻译、总结
- 一般性的问答和信息查询
- 普通编程辅助
- 大部分办公场景

> 💡 **核心结论**：80% 的日常需求，二梯队模型就能满足。没必要每个任务都用旗舰模型。

---

## 5.4 三梯队：轻量级模型

速度最快、价格最低（甚至免费），适合简单、高频的任务。

| 模型 | 出品方 | 核心优势 |
| --- | --- | --- |
| Gemini 3 Flash | Google | 极快、有免费档可用 |
| 豆包 Pro | 字节跳动 | 价格极低，语音交互和移动端体验好 |
| 腾讯元宝 | 腾讯 | 与微信/企微生态打通 |
| 文心 Speed | 百度 | 免费使用，日常轻量足够 |

### 什么时候用三梯队

- 简单问答和快速查询
- 日常聊天和轻度写作
- 批量处理简单任务（如批量翻译标题）
- 预算有限但用量大的场景

---

## 5.5 选型决策树

不知道自己该用哪个？按这个思路选：

```
你的任务难度高吗？（复杂推理、长文写作、专业代码）
├── 是 → 用一梯队（GPT-5.5 / Claude Opus 4.8 / DeepSeek-V4）
└── 否 → 你对中文质量要求高吗？
         ├── 是 → 用二梯队国产（GLM-5.1 / 通义 Qwen3.7 / Kimi）
         └── 否 → 你的用量大、预算低吗？
                  ├── 是 → 用三梯队（豆包 / Gemini Flash）
                  └── 否 → 用二梯队（GPT-5 mini / GLM-5.1）
```

---

## 5.6 能力评测：谁说了不算，数据说了算

业界有几个公认的评测基准，可以参考：

| 评测基准 | 测什么 | 说明 |
| --- | --- | --- |
| MMLU | 综合知识 | 57 个学科的选择题 |
| HumanEval | 编程能力 | 给函数写代码，测试通过率 |
| GSM8K | 数学推理 | 小学数学应用题 |
| C-Eval | 中文能力 | 专门测中文模型的基准 |
| Arena（LMSys） | 实战对比 | 真人盲测投票排名 |

> 💡 **最推荐参考 LMSys Arena 排名**——因为是真人盲测，最接近实际使用体验，不会被"刷分"误导。
